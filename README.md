# Heros App

Heros App es una aplicación construida con React + TypeScript que consume una API hecha en Nest.JS de héroes para mostrar una lista interactiva de personajes. La app permite explorar, buscar y gestionar héroes favoritos de manera sencilla e intuitiva.

## Tecnologias utilizadas

Typescript, React, ReactRouter, TanStack Query, TailwindCSS, Shadcn, ZOD.

[Server](https://github.com/Alexis-Alvarez-R/hero-server)

## Instalacion

Instalar dependecias con npm

```bash
  npm install
```

Levantar servidor Local

```bash
  npm run dev
```

## Variables de Entorno

Deberas colocar tu url base por ejemplo tu localhost

`VITE_API_URL`
